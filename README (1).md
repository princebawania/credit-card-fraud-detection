Credit Card Fraud Detection ML Pipeline
Overview

This project implements an end-to-end machine learning pipeline for credit card fraud detection. It includes data preprocessing, exploratory data analysis (EDA), anomaly detection, dimensionality reduction, clustering, and regression modeling to provide actionable insights for financial planning.

Features

Fraud Detection Pipeline: Developed using PCA, K-Means, and anomaly detection, achieving 97.1% accuracy.

Data Preprocessing: Automated workflows with pandas, NumPy, and StandardScaler for EDA and anomaly detection.

Regression Modeling: Built models to predict credit limits (R² = 0.74, RMSE = 19,000), assisting in financial planning decisions.

End-to-End Automation: From raw data to actionable insights.

Methodology

Data Preprocessing: Handled missing values, scaled features using StandardScaler, and automated EDA.

Dimensionality Reduction: Applied PCA to reduce feature space and improve model performance.

Clustering & Anomaly Detection: Used K-Means and anomaly detection to identify fraudulent transactions.

Regression Modeling: Built regression models to predict credit limits, providing actionable financial insights.

Evaluation Metrics: Accuracy for fraud detection and R² & RMSE for regression models.

Results

Fraud Detection Accuracy: 97.1%

Credit Limit Regression: R² = 0.74, RMSE = 19,000
